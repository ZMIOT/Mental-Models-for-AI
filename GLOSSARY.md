# Glossary

所有术语的核心解释以本文件为准。章节中可以展开，但不能改变核心含义。

## 术语使用规则

- 正文中文优先。
- 专用术语可以保留英文，但必须在本表中有稳定解释。
- 如果中文译名尚未确定，先保留英文，并在备注中标记“中文译名待确认”。
- 同一个术语在全书中只能有一个核心解释。

| 术语 | 核心解释 | 备注 |
| --- | --- | --- |
| Dataset | AI 对世界经验的结构化集合。 | 不只是文件夹，而是模型能学习到的世界样本。 |
| Sample | Dataset 中一次可学习的经验单位。 | 可以是一张图、一帧点云、一组多模态数据。 |
| Feature | 从样本中抽取或学习出的可用于判断的信号。 | 可以是人工设计，也可以由模型学习。 |
| Label | 用来告诉 AI 目标、类别、位置或正确答案的监督信号。 | 本质上是人或系统给经验附上的解释。 |
| Distribution | 数据在真实世界和采样集合中的出现方式。 | 用来理解训练数据和现场数据为什么可能不同。 |
| Coverage | Dataset 对真实世界变化范围的覆盖程度。 | 当前暂用英文术语，中文译名待确认。 |
| Augmentation | 用规则或变换扩展训练经验范围的方法。 | 不是凭空创造真实世界，只是补充可能变化。 |
| Generalization | AI 面对未见过但相关的新情况时仍能工作的能力。 | 受 Dataset、Model、Loss 和训练过程共同影响。 |
| Embedding | 把对象放进一个可比较空间后的表示。 | 用于相似性、聚类、检索和下游判断。 |
| Model | 一个可学习的映射函数。 | 从输入到输出，同时承载训练得到的经验。 |
| Loss | 用来衡量预测和目标之间差距的工程化信号。 | 它决定模型被推动往哪里改。 |
